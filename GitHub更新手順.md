# GitHub更新手順

## 方法1: VS CodeのGit機能を使用（推奨）

1. **VS Codeでソース管理パネルを開く**
   - 左側のソース管理アイコン（分岐マーク）をクリック
   - または `Ctrl + Shift + G`

2. **変更をステージング**
   - 「変更」セクションに変更されたファイルが表示されます
   - 各ファイルの横の「+」をクリックしてステージング
   - または「変更」の横の「+」をクリックしてすべてをステージング

3. **コミット**
   - 上部のメッセージボックスにコミットメッセージを入力
   - 例: `ログイン機能の改善と管理者ページへの自動遷移を追加`
   - 「✓」ボタンをクリックしてコミット

4. **プッシュ**
   - 左下の「同期」アイコンをクリック
   - または「...」メニューから「プッシュ」を選択

## 方法2: GitHub Desktopを使用

1. **GitHub Desktopを開く**
2. **変更を確認**
   - 左側に変更されたファイルが表示されます
3. **コミット**
   - 下部の「Summary」にコミットメッセージを入力
   - 「Commit to main」をクリック
4. **プッシュ**
   - 「Push origin」をクリック

## 方法3: Git Bashを使用

Git Bashがインストールされている場合：

```bash
git add .
git commit -m "ログイン機能の改善と管理者ページへの自動遷移を追加"
git push
```
