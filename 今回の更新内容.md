# 今回の更新内容

## 更新日: 2025年1月

### 主な変更点

1. **予定データをユーザーごとに分離**
   - `events` → `events_${currentUserId}`
   - `completedEvents` → `completedEvents_${currentUserId}`
   - 各アカウントが自分の予定のみを表示・管理できるように変更

2. **予定履歴をユーザーごとに分離**
   - `eventLocationHistory` → `eventLocationHistory_${currentUserId}`
   - チェックボックスにチェックを入れた予定のみが履歴に保存されるように変更

3. **calendar.htmlのUI改善**
   - ログアウトボタンをアカウントボタンに変更
   - アカウント名を表示
   - アカウントメニューを追加（基本情報、統計情報、終了した予定一覧、ログアウト）
   - 「終了した予定一覧を見る」ボタンを削除（アカウントメニューに統合）

4. **管理者ダッシュボードの改善**
   - アカウント削除機能を追加
   - 各アカウントカードに削除ボタンを追加

5. **index.htmlの改善**
   - ログインボタンを追加
   - ログインセクションのUI改善
   - 管理者メールアドレスでログイン時に自動的に管理ページに遷移

6. **予定履歴の削除機能**
   - 各履歴項目に削除ボタンを追加

7. **check.htmlの改善**
   - 現在位置を人のアイコンで表示
   - 現在位置の周りに円を表示

8. **その他の修正**
   - 予定の「選択」ボタンを「確認」に変更
   - 「確認」ボタンでcheck.htmlに直接遷移
   - 管理者専用ログインページ（admin.html）を削除

## 変更されたファイル

- `js/calendar.js` - 予定データのユーザー分離、予定履歴の表示修正
- `js/set.js` - 予定データの保存をユーザーごとに分離
- `js/check.js` - 予定結果の記録をユーザーごとに分離、現在位置表示の改善
- `js/completed.js` - 終了した予定の表示をユーザーごとに分離
- `calendar.html` - アカウントメニューの追加、UI改善
- `index.html` - ログイン機能の改善、UI改善
- `admin-dashboard.html` - アカウント削除機能の追加
- `admin.html` - 削除（管理者メールアドレスで自動遷移するため不要）
