# Live Server 正しい使い方

## ⚠️ 重要な注意 ⚠️

`0.0.0.0` は**サーバー側の設定**です。ブラウザで直接 `http://0.0.0.0:5502` にアクセスすることは**できません**。

## ✅ 正しいアクセス方法

### PCのブラウザから
```
http://localhost:5502
```
または
```
http://127.0.0.1:5502
```

### iPhoneから
```
http://[PCのIPアドレス]:5502
```
例：`http://192.168.1.100:5502`

**PCのIPアドレスを確認する方法：**
- `GET_IP.bat` をダブルクリック

---

## 設定について

`.vscode/settings.json` の設定：
```json
{
    "liveServer.settings.port": 5502,
    "liveServer.settings.host": "0.0.0.0"
}
```

この設定の意味：
- **host: "0.0.0.0"** = すべてのネットワークインターフェースでリッスン
- これにより、iPhoneからもアクセス可能になります
- ただし、PCのブラウザでは `localhost` を使います

## トラブルシューティング

### 問題: iPhoneから接続できない

#### ステップ1: PCで動作確認
1. PCのブラウザで `http://localhost:5502` を開く
2. ページが表示されることを確認

#### ステップ2: IPアドレスを確認
1. `GET_IP.bat` をダブルクリック
2. 表示されたIPアドレスをメモ

#### ステップ3: iPhoneで接続
1. PCとiPhoneが**同じWi-Fi**に接続されているか確認
2. iPhoneのSafariで `http://[ステップ2で確認したIP]:5502` を開く
   - 例：`http://192.168.1.100:5502`

#### ステップ4: ファイアウォールを確認
もしまだ接続できない場合：
1. Windowsのファイアウォール設定を開く
2. ポート5502の受信規則を追加（Live Server トラブルシューティング.md参照）

---

## まとめ

- ✅ PCからは `http://localhost:5502`
- ✅ iPhoneからは `http://[PCのIPアドレス]:5502`
- ❌ `http://0.0.0.0:5502` は使用しない（サーバー設定のみ）
