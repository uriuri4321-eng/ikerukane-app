# 友達に使ってもらう手順

## 📋 事前準備（あなたが行うこと）

### 1. Firebaseプロジェクトの設定

1. [Firebase Console](https://console.firebase.google.com/)にアクセス
2. プロジェクトを作成（まだの場合）
3. `Firebase設定手順.md`に従って設定を完了
4. `js/firebase-config.js`に設定値を入力

### 2. GitHub Pagesにデプロイ

1. 変更をGitHubにプッシュ
   ```bash
   git add .
   git commit -m "Firebase統合とGoogleアカウントログイン追加"
   git push
   ```

2. 数分待ってから公開URLを確認
   - `https://uriuri4321-eng.github.io/ikerukane-app/`

### 3. Firebase Authenticationの設定確認

- Googleアカウントログインが有効になっているか確認
- メール/パスワードログインも有効にしておくと便利

## 📱 友達が使う手順

### 方法1: GitHub PagesのURLを共有（推奨）

1. **友達にURLを送る**
   ```
   https://uriuri4321-eng.github.io/ikerukane-app/
   ```

2. **友達の操作**
   - URLを開く
   - 「Googleアカウントで登録」または「アカウント登録」をクリック
   - Googleアカウントでログイン（推奨）またはメール/パスワードで登録
   - 予定を設定して使用開始

### 方法2: QRコードで共有

1. **QRコードを生成**
   - オンラインツール（例：https://qr-code-generator.com/）を使用
   - URL: `https://uriuri4321-eng.github.io/ikerukane-app/`
   - QRコードを画像として保存

2. **友達に送る**
   - メッセージアプリでQRコード画像を送る
   - 友達がスマホでQRコードを読み取る
   - 自動的にURLが開く

## 🔐 ログイン方法の説明

友達に以下の2つの方法を説明してください：

### 1. Googleアカウントでログイン（簡単・推奨）
- 「Googleアカウントでログイン」ボタンをクリック
- Googleアカウントを選択
- 自動的にログイン完了

### 2. メール/パスワードでログイン
- メールアドレスとパスワードを入力
- 「ログイン」ボタンをクリック

## 📊 管理者として確認する方法

1. `https://uriuri4321-eng.github.io/ikerukane-app/admin-dashboard.html`にアクセス（管理者メールアドレスでログイン後、自動的に遷移します）
2. ユーザー名: `admin` / パスワード: `admin123`でログイン
3. 全ユーザーの登録情報と統計を確認

## ⚠️ 注意事項

### 友達に伝えること

1. **初回使用時**
   - 位置情報の許可を求められたら「許可」を選択
   - ブラウザの設定で位置情報を許可する必要があります

2. **iPhoneで使用する場合**
   - Safariブラウザを使用
   - 位置情報の許可が必要
   - HTTPSでアクセスしていることを確認（GitHub Pagesは自動でHTTPS）

3. **ログイン状態の保持**
   - 同じブラウザ・デバイスでは自動的にログイン状態が保持されます
   - ブラウザのデータを削除するとログイン情報も消えます

### あなたが確認すること

1. **Firebase Consoleでユーザーを確認**
   - Firebase Console → Authentication → Users
   - 登録されたユーザー一覧を確認

2. **Firestoreでデータを確認**
   - Firebase Console → Firestore Database
   - `users`コレクションにユーザー情報が保存されているか確認

3. **管理者ダッシュボードで確認**
   - 全ユーザーの情報と統計を確認

## 🚀 テスト手順

1. **自分でテスト**
   - スマホでURLを開く
   - Googleアカウントで登録
   - 予定を設定して動作確認

2. **友達にテストしてもらう**
   - URLを共有
   - 1週間使用してもらう
   - フィードバックを収集

3. **管理者ダッシュボードで確認**
   - 登録されたアカウント数を確認
   - 各ユーザーの統計情報を確認
