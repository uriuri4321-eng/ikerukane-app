============================================
   Live Server すぐに使う方法（超簡単）
============================================

【重要なポイント】
設定ファイルはそのままで大丈夫です！
0.0.0.0 はサーバーの設定で、ブラウザからは使えません。

============================================
ステップ1: Live Serverを起動
============================================
1. VS Codeで index.html を右クリック
2. 「Open with Live Server」を選択
3. PCのブラウザが自動で開く（localhost）

============================================
ステップ2: PCのIPアドレスを確認
============================================
GET_IP.bat をダブルクリック
例：192.168.1.100 と表示されたらメモ

============================================
ステップ3: iPhoneからアクセス
============================================
iPhoneのSafariで以下を開く：

   http://[PCのIPアドレス]:5502

例：http://192.168.1.100:5502

============================================
【補足】
============================================
- PCのブラウザからは → http://localhost:5502
- iPhoneからは → http://[PCのIP]:5502
- 両方とも同じサーバーにアクセスしています
- 設定ファイル（.vscode/settings.json）は変更不要です
